# Technology Stack

## Core Language & Runtime
- **ArkTS:** A TypeScript-based language optimized for the HarmonyOS environment, providing high performance and a declarative UI development experience.
- **Node.js:** Required for build tools and dependency management.

## Platform & Frameworks
- **HarmonyOS / OpenHarmony (Stage Model):** The primary application model providing a clear lifecycle management and resource isolation.
- **ArkUI:** A declarative UI framework for building fluid and high-performance user interfaces.

## Build & Dependency Management
- **Hvigor:** The official build tool for HarmonyOS, providing efficient task-based builds.
- **ohpm (OpenHarmony Package Manager):** Used for managing project dependencies and modules.

## Testing & Quality Assurance
- **Hypium:** The standard testing framework for HarmonyOS, used for both unit testing and UI automation.
- **Hamock:** A mocking framework used in conjunction with Hypium for isolating units under test.
- **Code Linter:** Integrated tool for ensuring code quality and adherence to ArkTS style guidelines.

## Target Devices
- **Phone & Wearable:** Optimized for various form factors as defined in the module configuration.
